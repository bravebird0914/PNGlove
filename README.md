# PNGlove

あらゆる画像フォーマットを **PNG に変換する** ブラウザアプリ。

`index.html` をダブルクリックするだけで動く。インストール不要。サーバー不要。ファイルはどこにも送信されない。

## 対応フォーマット

HEIC / HEIF / TIFF / WebP / AVIF / BMP / GIF / ICO / SVG / JPEG など → **PNG**

> iPhone で撮った HEIC 写真を Windows でも見られるようにする、というのが最小の目的。

## 使い方

1. `index.html` をブラウザで開く
2. ファイルをドラッグ＆ドロップ
3. 「変換する」

## 注意

初回起動時はインターネット接続が必要（変換ライブラリを CDN から読み込む）。  
以降はオフラインでも動作する。
